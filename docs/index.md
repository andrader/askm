# jup 🚀

Agent Skills Manager. Lightweight Python-based command-line tool designed to manage and synchronize "agent skills" across various AI assistant directories.

## Installation

```bash
pip install jup
```

## Quick Start

```bash
jup add gh:owner/repo/skill-name
jup sync
```

## Features

- **Sync everywhere**: Manage skills for Claude, ChatGPT, and other agents in one place.
- **Easy discovery**: Find new skills on GitHub.
- **Interactive**: Selectively sync only what you need.
