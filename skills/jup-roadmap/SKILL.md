# jup Roadmap & Ideation Assistant 🚀

This skill helps AI agents iterate on the `jup` roadmap, brainstorm new features, and refine existing ideas.

## How to Use This Skill

- **Read the Current Roadmap**: Start by reading `backlog/roadmap.md` to understand the current state and future plans.
- **Analyze Backlog Items**: Evaluate existing items for feasibility, impact, and clarity.
- **Brainstorm New Features**: Suggest features that align with `jup`'s goal of simplified agent skill management.
- **Refine Ideas into Specs**: Take a broad idea and break it down into a technical spec with clear requirements and success criteria.

## Core Ideation Principles

- **Simplicity First**: `jup` aims to be a lightweight and intuitive tool. Avoid unnecessary complexity.
- **Agent-Centric Design**: Think about how AI agents (like yourself) will use `jup` and what features would make their job easier.
- **Local-First Reliability**: Prioritize features that work reliably on local environments and respect the user's local configuration.
- **Seamless Integration**: Ensure new features integrate well with existing commands and supported agent directories.

## Deliverables

- **Roadmap Updates**: Propose additions, removals, or priority changes to `backlog/roadmap.md`.
- **Feature Proposals**: Write concise summaries of new feature ideas.
- **Technical Specs**: Draft detailed specifications for the most promising ideas.
- **User Feedback Analysis**: Analyze potential user needs and suggest improvements based on those needs.
